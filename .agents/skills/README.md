# 项目 UI 设计 Skills

这些 skills 用于后续设计或实现 `apps/web` 的 AI 产品/SaaS 前端体验。它们只提供可复用设计约束与检查清单，不自动触发构建。

## 技能索引

- `designing-ai-trustworthy-interfaces`: AI 输出、建议、评分、排序、生成结果的信任校准。
- `designing-ai-feedback-control`: AI 反馈、人工接管、撤销、重试、偏好重置。
- `designing-saas-operational-workspaces`: SaaS 操作台、Run Viewer、任务流、详情页布局。
- `designing-saas-data-tables`: 表格、筛选、排序、批量操作、空状态与异常状态。
- `designing-saas-design-systems`: token、组件状态、可访问性、密度、交付约束。
